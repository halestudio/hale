Groovy Join

With Groovy Join function, you could join two source schemas based on the conditions entered. This function allows user to add the entities for the function, gives the ability to configure the join order of the added source types and asks for the conditions on which two types should be joined. You can add multiple join conditions. The joined structures can be viewed in the show source structure. Based on the conditions given for the join, structure will be appeared in the hierarchical manner. You can select any property or type and it will give the examples stating the usage of the function. Helper Function Specification

This allows user to get the specification of the helper function. This is available on click of Show functions item. The tray lists down all the available helper functions. When an individual function is clicked in this tray, the specification about that function will be displayed in the tray located at bottom right. This specification includes :

  • Description about the function.
  • The parameters in the function, description of each parameter and its default value.
  • If function takes more than one parameters and if the default value is not given then it will display ? to show that default value is unknown.
  • Description about the return type of the function.

The helper functions with specification is also appeared in completion proposal. The code starts with _. and then pressing ctrl + space would give the completion proposal. If you select a particular function and highlight it, it would also show some additional information regarding the function. The additional information include description of the function, what parameters the function takes, description of each parameter with their default values, and the return of the function.
