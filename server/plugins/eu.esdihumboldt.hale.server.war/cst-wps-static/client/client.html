HUMBOLDT cst-wps web client

The HUMBOLDT Conceptual Schema Transformer (CST) is a Web Processing Service that is able to apply a schema transformation to a geodata set (using a specific source Application Schema (A)) in order to provide a geodata set (expressed in a specific target Application Schema B). A schema mapping between schema A and schema B has to be defined beforehand in order to accomplish the transformation.

You can access the OpenGeospatial Consortium Web Processing Service 1.0.0 capabilities of this installation by following this link.

To use the service, either upload files for each of the parameters desribed below or provide URLs to locations where the applicable resources can be accessed.

Source Data

A GML (2.1, 3.1 or 3.2) source that contains the source data to be transformed. Important: All source data sets need to be of the same GML Application Schema!

Source XML Schema Definition

XSD used to define the source data's GML Application Schema. This input needs to be the "primary" GML Application Schema. If it has dependencies to other schemata, they should be either accessible on the web or provided through AuxiliarySourceXmlSchemaArchive.

Source XML Schema Dependencies Archive

ZIP archive containing source schema dependencies (AuxiliarySourceXmlSchemaArchive). This parameter is optional if your source schema has no dependencies to other schemas or if all included schemas are available online.

Target XML Schema Definition

XSD used to define the resulting target data's GML Application Schema. This input needs to be the "primary" GML Application Schema. If it has dependencies to other schemata, they should be either on the web or provided through AuxiliaryTargetXmlSchemaArchive.

Target XML Schema Dependencies Archive

ZIP archive containing target schema dependencies (AuxiliaryTargetXmlSchemaArchive). This parameter is optional if your target schema has no dependencies to other schemas or if all included schemas are available online.

Ontology Mapping Language File

OML file that contains a mapping between the primary source and the primary target XSD/GML Application Schema..
