Saving your work

HALE allows you to save your mapping, together with the configuration which schemas and source data sets are used in an alignment project. The project also includes additional settings, e.g. the list of mapping relevant types and the map styles. You can continue working on a project later on, by loading it again in the application.

To save a project, use the Save Alignment Project as... command in the File menu or the application tool bar. Recently saved or loaded projects are available in the File menu and on the welcome page for quick access, for loading another project you can use the Open Alignment Project command.
