Properties view

The properties view displays detailed information about the currently selected object. Depending on the type of object, a different set of properties is displayed. The object types for which this information is available in hale»studio are the following:

  • Schema elements (e.g. selection from the Schema Explorer or Type hierarchy view)
  • Mapping cells (e.g. as selected in the Alignment view)
  • Functions (e.g. selection from the Functions view)
  • Reports (e.g. as selected in the Report List)
