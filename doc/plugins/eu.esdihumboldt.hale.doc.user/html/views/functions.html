Functions view

The Functions view shows the available transformation functions, which can be used to define relations between source and target schema elements. They are listed according to their associated function category.

Further information on a selected function will be displayed in the Properties view, e.g. the function description or a graph depicting the inputs, outputs and parameters of the function. The functions reference in this help also holds a detailed description of the available functions.
