Data As XLS Export

The data (source and target) can be exported as an excel file. The type presented as the sheet's name and attribute names as the headline.

Exporting data which is present in a complex schema is possible if the "Solve nested properties" option is checked during the export process. The attribute's name is the path from the type to the attribute. person.address.city for example, with the column containing their instance value of the city attribute.

At the current state it is not possible to choose between different types. If more than one type is is present only one is exported.
