HTML Mapping Documentation

Once you have defined an alignment, you can export a HTML mapping documentation on it. The mapping documentation lists per type relation all associated mapping cells, with a graph depicting the cell and a cell explanation, taking into account the inputs, outputs and parameters. The mapping documentation on the completed alignment for the example project from the Get started with hale»studio guide is provided as an example.

To create the mapping documentation for your project, export the alignment...
