New in 2.5.0

Several bugfixes Based on the feedback from the use of the release candidate we could identify and solve several bugs to make HALE ready for the stable release. For instance, the issues with the map view on Linux/GTK have been resolved and the performance of the XML/GML encoding has been improved.
